#DocVault Project
#Overview

DocVault is a cloud-enabled document management application built with .NET 8 Web API, Angular 17 frontend, and Azure cloud services. The project allows users to upload documents, store them in Azure Blob Storage, and manage metadata in Cosmos DB.
